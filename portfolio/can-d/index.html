Controller Area Networks (CAN busses) are commonly found in a various everyday applications. One of the most common uses of these networks is the networking of automotive electronic control units (ECUs).

ECUs are the small computers that control the systems that are found within automotive vehicles. Modern cars are made up of numerous ECUs and they all communicate with each other by sending messages over the CAN bus.

A CAN data logger is a necessary tool for any developers or engineers working with the a CAN bus. This tool allows developers to thoroughly analyse and understand the data that is passed between the ECUs in their vehicle. Current solutions available on the market are not affordable and typically only support the Windows operating system.

We created a data logger that is robust, simple, and affordable. Additionally, a cross platform, open-source user interface desktop application was created to allow users of our project to view and understand the data that they have logged.
